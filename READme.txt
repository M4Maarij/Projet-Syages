					SYAGES
Le Système Automatique de Gestion des Etudiants-Stagiaires est une plateforme
 web permettant la gestion administrative et pédagogique des étudiants-stagiaires
 en formation DEAU. La formation leur permet d’obtenir un diplôme de niveau IV 
équivalent au baccalauréat. Cette plateforme de type Pronote sert à faciliter 
l'administration des étudiants à la fois pour les professeurs, les secrétaires : 
inscriptions, formations, notes, absences.




---> si le css ou les images ne fonctionnent pas regardez bien si vos dossiers img et css 
sont bien dans le document root ("www" pour wampp et "htdocs" pour xampp).

---> veuillez attribuer le path de chaque image dans le dossier img à son emplacement 
respectif dans les fichiers css/html pour un affichage correct.
